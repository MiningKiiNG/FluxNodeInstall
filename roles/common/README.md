Common
=========

An Ansible role to install common packages and configure common settings for FluxNodes.


Requirements
------------

This role requires Ansible 2.0 or higher.

Role Variables
--------------

```yaml
# These are from the vars.yml file and come with defaults
ip:
  url1:
  url2:
```

Dependencies
------------

None.
